# Сокращение ссылок с помощью VK

Утилита для работы с VK API: сокращает длинные ссылки в vk.cc и показывает количество кликов по сокращённым.

### Как установить

Для получения данных от API VK потребуется сервисный ключ – “токен”. Зарегистрируйтесь в [социальной сети ВК](https://vk.ru/), перейдите в раздел [для разработчиков](https://dev.vk.com/ru) и создайте новое приложение. В меню разработки будут доступны ключи доступа. Вы увидите сервисный ключ приложения на подобии такого: 
```
82a02da882a02da882a02da8a981b7f3cc882a082a02da8e4af9c41e8551329276dde72. 
```
Ключ сохраните в файле `.env`.

Python3 должен быть уже установлен.
Затем используйте `pip`  (или `pip3`, если есть конфликт с Python2) для установки зависимостей. Для этого в командной строке необходимо ввести команду:
```
pip install -r requirements.txt
```
Рекомендуется использовать [модуль venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.
